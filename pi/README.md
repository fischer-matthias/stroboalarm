# pi server application

## Clone project
```
git clone --recursive git@github.com:fischer-matthias/stroboalarm.git
```

## Requirements
- [WiringPi](https://projects.drogon.net/raspberry-pi/wiringpi/download-and-install/)

## Install
```
npm install
npm run build
```

## Run server
```
npm start
```
